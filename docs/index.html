PoffertjesUI

Open Source CSS Framework by Netail

Introduction

PoffertjesUI is a CSS framework inspired by Samsung's OneUI, Spotify and iOS.

Theming

PoffertjesUI support themes, which are used by adding a theme attribute into the html tag. For example: <html theme="dark">.
Themes override the default "Light theme" CSS variables, which can be found here. (Keep in mind that the theming files can change as this CSS framework is still in development)
The default font for this CSS framework is "Open Sans", which has to be imported yourself using this link.
